# 規制カテゴリ向けガードレール

たばこ、加熱式たばこ、ニコチン、喫煙、ベイパー、酒類、医療、金融など、年齢制限や法務確認が必要な商材を扱う場合に読む。

## 必須チェック

- 対象が成人のみの場合、その前提を明示する。若年層、学生、未成年、学校、家族全体の利用を示唆しない。
- 健康、安全、禁煙、医療、リスク低減に関する訴求は、法務/オリエン資料で明示承認された文言以外は使わない。
- 商品が無害、クリーン、健康的、安全、どこでも使えると誤認される表現を避ける。
- 未成年に見える人物、キャラクター、子ども向けのビジュアル、菓子のような表現、若年カルチャーの強い記号を使わない。
- オリエンで明示承認されていない、法務上センシティブな使用シーンを描かない。
- 価格、割引、発売日、販売場所、キャンペーン条件、商品機能、受賞、ランキング、法定表示を勝手に作らない。
- ブランド名と商品名は、提供資料にある正式表記を使う。
- ロゴ比率、クリアスペース、指定色、最小サイズ、ロックアップ/デバイスロゴの組み合わせを守る。

## 推奨コピー方向

- 提供資料で裏づけがある範囲で、成人向けの味わい、デザイン、触感、デバイス形状、サービス体験、キャンペーン参加方法を扱う。
- 断定的な効能や優位性ではなく、資料に基づく説明表現を使う。
- 落ち着いた、上質で、大人向けのトーンにする。

## QA追加項目

規制カテゴリの場合、`qa_report.md` に以下を含める。

- 成人向け対象チェック
- 未承認の健康/安全/禁煙/リスク低減訴求チェック
- 若年層訴求のビジュアル/文言チェック
- 商品機能/価格/販売条件などの正確性チェック
- ロゴ/ブランドロックアップ準拠チェック
